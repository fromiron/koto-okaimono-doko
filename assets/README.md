# Image assets

Generated visual assets for `koto okaimono doko`.

Brand text:

- English: `koto okaimono doko`
- Japanese: `こうとうお買い物どこ`

These assets are original, app-specific illustrations and do not include official logos, mascots, or copied visuals from the coupon program.

| File | Use |

| `brand-style-guide.svg` | Visual direction, palette, app icon concept, marker/badge reference |
| `illustration-map-empty.svg` | Map home empty/search state |
| `illustration-store-detail.svg` | Store detail / bottom sheet illustration |
| `illustration-mall-group.svg` | Mall/facility group state |
| `illustration-filter.svg` | Filter modal illustration |
| `illustration-location-permission.svg` | Location permission state |
| `illustration-offline-cache.svg` | Offline / cached dataset state |
| `illustration-dataset-update.svg` | Settings / dataset update / transparency state |
