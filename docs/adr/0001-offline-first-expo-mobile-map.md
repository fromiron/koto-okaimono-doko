# Offline-first Expo mobile map

The app is an Android-first, iOS-compatible Expo and React Native app that uses Expo Router, React Native Maps, and local SQLite data instead of a backend. Store data, search, and details must work from a bundled Dataset without network access; map tiles may still depend on the platform map provider cache or network. MVP excludes EAS Update, background location, analytics, Firebase, Supabase, Mapbox, and Google Places so the first release stays small and privacy boundaries remain clear.
